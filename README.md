# DSL_winter_project 
### News Article Topic Classification (Winter 2026 – DS & ML Lab)
Machine learning project for automatically classifying online news articles into predefined categories (e.g. International, Business, Technology, Sports) using textual content and metadata. Built on a ~100k-article dataset provided by Politecnico di Torino, with a full pipeline including preprocessing, feature extraction, model training, validation, and evaluation using Macro F1-score.
